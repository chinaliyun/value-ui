# 设置宽高尺寸

Scene 组件支持快速为元素设置尺寸, 纯数字会默认当做px单位, 支持px|rem|em|vh|vw|%等单位

设置宽度200px

设置宽度50%

设置宽度20vw

设置高度50px

设置高度4em

设置高度10vh
<template>
  <div class="code_wrapper">
    <var-scene w="200" class="line">设置宽度200px</var-scene>
    <br />
    <var-scene w="50%" class="line">设置宽度50%</var-scene>
    <br />
    <var-scene w="20vw" class="line">设置宽度20vw</var-scene>
    <br />
    <var-scene h="50" class="line">设置高度50px</var-scene>
    <br />
    <var-scene h="4em" class="line">设置高度4em</var-scene>
    <br />
    <var-scene h="10vh" class="line">设置高度10vh</var-scene>
  </div>
</template>
<style lang="scss" scoped>
.line {
  background-color: rgb(238, 238, 238);
  height: 30px;
}
</style>

四个方向都被设置为20px的内边距

顶部内边距20px


上下两侧内边距20px

# 允许子元素扩展宽度

fill属性, 该 具备 flex 属性的元素的所有子元素都会被添加flex-grow:1的样式;

100px宽的元素
100px宽的元素

100px宽的元素
200px宽的元素 
<template>
  <div class="code_wrapper">
    <var-scene class="line" flex fill>
      <var-scene class="child"> 100px宽的元素 </var-scene>
      <var-scene class="child"> 100px宽的元素 </var-scene>
    </var-scene>
    <br />
    <var-scene class="line" flex fill>
      <var-scene class="child" w="100"> 100px宽的元素 </var-scene>
      <var-scene class="child" w="200"> 200px宽的元素 </var-scene>
    </var-scene>
  </div>
</template>

.line {
  background-color: rgb(240, 240, 240);
  height: 50px;

    border: 1px solid rgb(161, 152, 238);
  }
}
</style>

# 允许元素本身扩展宽度

grow属性会为元素本身添加flex-grow:1的属性, 也可以为grow属性赋值, 比如grow=2相当于给元素添加了flex-grow:2的样式属性

100px宽的元素
100px宽的元素

100px宽的元素
100px宽的元素 
<template>
  <div class="code_wrapper">
    <var-scene class="line" flex>
      <var-scene w="100" class="child" grow> 100px宽的元素 </var-scene>
      <var-scene w="100" class="child" grow> 100px宽的元素 </var-scene>
    </var-scene>
    <br />
    <var-scene class="line" flex>
      <var-scene w="100" class="child" grow="1"> 100px宽的元素 </var-scene>
      <var-scene w="100" class="child" grow="2"> 100px宽的元素 </var-scene>
    </var-scene>
  </div>
</template>
<style lang="scss" scoped>
.line {
  background-color: rgb(240, 240, 240);
  height: 50px;
  .child {
    border: 1px solid rgb(161, 152, 238);
  }
}
</style>

# 禁止元素扩展/缩放

fixed属性可以设置元素在 flex 布局中始终保持自己的宽度, 该属性会为元素添加flex-shrink:0;flex-grow:0的样式属性

100px宽的元素
100px宽的元素
200px宽的元素 
<template>
  <div class="code_wrapper">
    <var-scene class="line" flex fill>
      <var-scene class="child" w="100"> 100px宽的元素 </var-scene>
      <var-scene class="child" w="100"> 100px宽的元素 </var-scene>
      <var-scene class="child" w="200" fixed> 200px宽的元素 </var-scene>
    </var-scene>
  </div>
</template>
<style lang="scss" scoped>
.line {
  background-color: rgb(240, 240, 240);
  height: 50px;
  .child {
    border: 1px solid rgb(161, 152, 238);
  }
}
</style>

# 网格化布局

前缀 尺寸
xs media query > 768
sm media query > 992
md media query > 1190
lg media query > 1440

下面的案例中

可视区域大于1440宽时, 单个元素占据1/6的宽度
可视区域大于1190宽时, 单个元素占据2/6的宽度
可视区域大于960宽时, 单个元素占据3/6的宽度
可视区域大于768宽时, 单个元素占据4/6的宽度
可视区域小于768宽时, 单个元素占据6/6的宽度
