# Avatar 头像

# 基础用法

<template>
  <var-avatar :src="src"></var-avatar>
</template>
<script>
export default {
  name: "AvatarBase",
  data() {
    return {
      src: require("./images/1.jpg"),
    };
  },
};
</script>

# 自定义尺寸

默认的头像尺寸是 80px, 可以通过 size属性自定义尺寸

<template>
  <var-scene flex middle>
    <var-avatar :src="src" size="100"></var-avatar>
    <var-avatar :src="src" size="120"></var-avatar>
    <var-avatar :src="src" size="140"></var-avatar>
  </var-scene>
</template>
<script>
export default {
  data() {
    return {
      src: require("./images/1.jpg"),
    };
  },
};
</script>

# props

prop 默认值 说明
src null 头像地址
default null 默认头像地址, 如果 src 加载失败, 且 default 有值的情况下, 会加载 default 头像
size 80px 定义头像的尺寸,
alt null 同 img元素的alt属性
